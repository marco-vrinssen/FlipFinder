# Flipper

Auctionator companion for finding arbitrage deals in World of Warcraft.

## Features

- **Arbitrage Detection**: Notifies you of profitable spread opportunities.
- **Auctionator Integration**: Works seamlessly as a companion to Auctionator.
- **Live Price Scanning**: Monitors the first 5 price brackets for 10%+ profit spreads.

## Installation

1. Download the latest version.
2. Extract the files to your `Interface/AddOns` folder.
3. Restart World of Warcraft.
